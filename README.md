# Test on your machine
```sh
git clone https://github.com/daxeel/Instagram-Algorithm.git
cd Instagram-Algorithm
```
```py
python insta-cli.py [INSTAGRAM_USERNAME]
```
For example,
```py
python insta-cli.py daxeel_soni
```
